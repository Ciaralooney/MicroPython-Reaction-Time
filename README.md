This is a reaction game built with micro python. It uses an LED on the board and the I2C LCD screen to transmit messages to the user. 

<img width="686" height="267" alt="image" src="https://github.com/user-attachments/assets/dde1d25a-6f0a-42da-822a-3fb54a278d93" />

It displays the reaction time and a message depending on the how fast the user is.
